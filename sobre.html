¿Qué es SPG?

SPG (Sistema de Proyectos para Constructora) es una solución digital creada para ayudar a pequeñas empresas constructoras a gestionar sus proyectos de forma eficiente.

Objetivos del Proyecto

  • Organizar los proyectos por estado: planificación, ejecución y finalización.
  • Facilitar el seguimiento de tareas y recursos.
  • Ofrecer una interfaz clara y adaptable a dispositivos móviles.

Aprendizaje con GIT

Este proyecto también sirvió como práctica para aprender el uso de Git, un sistema de control de versiones que permite organizar los cambios, trabajar en equipo y mantener un historial limpio del desarrollo.
